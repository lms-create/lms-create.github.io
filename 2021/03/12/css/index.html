前端知识体系（CSS）

盒子模型

w3c标准盒模型（content-box）

content只包括content，没有其余内容。

IE盒模型（border-box）

content包括padding和border。

flex布局

容器属性

flex-direction 布局的方向

flex-wrap 是否换行，换行的顺序

flex-flow 上面2个属性的简写

justify-content 水平方向的排列

align-items 竖直方向的排列

align-content 多条轴线的content 竖直方向的排列

项目属性

order 属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。

flex-grow 属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

flex-shrink 属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

flex-basis

flex 以上3个属性的缩写

align-self 允许单个项目，有自己的排列方式。

position定位

相对定位

相对于自己一开始的位置进行定位。

绝对定位

相对于开启了定位的父级元素的位置进行定位。

固定定位

相对于浏览器窗口进行定位。

<html>
	<head>
		<meta charset="UTF-8">
		<title></title>
		<style type="text/css">
			*{
				margin:0;
				padding:0;
			}
			.wrap{
				display:flex;
				flex-direction:row;
				margin-top:20px;
			}
			.center{
				width:800px;
				text-align:center;
				background:#ccc;
			}
			.left,.right{
	        flex-grow: 1;
	        line-height: 30px;
	        background:red;
	     }
		</style>
	</head>
	<body>
		<div class="wrap">
			<div class="left">left</div>
			<div class="center">center</div>
			<div class="right">right</div>
		</div>
	</body>
</html>

a标签样式的LoVe/HAte原则

1 a:link，定义正常链接的样式；
2 a:visited，定义已访问过链接的样式；
3 a:hover，定义鼠标悬浮在链接上时的样式；
4 a:active，定义鼠标点击链接时的样式。
定义样式一定要按照这个顺序定义。
