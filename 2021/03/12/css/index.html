前端知识体系（CSS）（一）

盒子模型

w3c标准盒模型（content-box）

content只包括content，没有其余内容。

IE盒模型（border-box）

content包括padding和border。

flex布局

容器属性

flex-direction 布局的方向

flex-wrap 是否换行，换行的顺序

flex-flow 上面2个属性的简写

justify-content 水平方向的排列

align-items 竖直方向的排列

align-content 多条轴线的content 竖直方向的排列

项目属性

order 属性定义项目的排列顺序。数值越小，排列越靠前，默认为0。

flex-grow 属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。

flex-shrink 属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。

flex-basis

flex 以上3个属性的缩写

align-self 允许单个项目，有自己的排列方式。

position定位

相对定位

相对于自己一开始的位置进行定位。

绝对定位

相对于开启了定位的父级元素的位置进行定位。

固定定位

相对于浏览器窗口进行定位。

清除浮动 （父元素因为子级元素的浮动而导致的内部高度为0的问题）

1.在父元素中添加空div clear:both

2.对父元素添加伪类

3.触发父元素BFC（overflow属性）

BFC相关（块级格式化上下文）

1.避免某元素被浮动元素覆盖

2.清除浮动

3.阻止外边距重叠

行内元素和块级元素

行内元素就是占一整行的元素。

块级元素就是一行内可以有多个。

div垂直居中

1.flex

2.absolute+transform

3.absolute+margin

4.margin auto

5.display: table-cell

6.cacl()
